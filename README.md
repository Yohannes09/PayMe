# PayMe

Direct payment service. 

## Features (Under Development)
- **API Gateway** 
- **Rework to Authentication Service**
- **Gateway, Authentication Service, and Token Service integration** 
- **User service** 


## Tech Stack

- **Backend:** 
  - **Framework:** SpringBoot
  - **Microservice Architecture:** Each service is containerized using Docker.
  - **ORM:** Hibernate.
  - **Asynchronous Communication:** Kafka 
  - **Deploy strategy:**

- **Database:**
  - PostgreSQL.

- **Frontend:**
  - HTML, CSS, and JavaScript.

- **Authentication:**
  - JWT, with plans to integrate OAuth 2.0.
